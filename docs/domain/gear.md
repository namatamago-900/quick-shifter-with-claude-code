# ギア系ドメイン知識 — Ducati 900SS (1995年式)

## 1. 一次減速比（Primary Drive Ratio）

| 車種 | ギア構成 | 減速比 |
|------|----------|--------|
| **900 S.S.** | **62 / 31** | **2.000** |

---

## 2. 内部ギア比（Internal Gear Ratios）

| ギア | 歯数（ドライブ / ドリブン） | ギア比 |
|------|--------------------------|--------|
| **1速** | 37 / 15 | **2.467** |
| **2速** | 30 / 17 | **1.765** |
| **3速** | 27 / 20 | **1.350** |
| **4速** | 24 / 22 | **1.091** |
| **5速** | 23 / 24 | **0.958** |
| **6速** | 24 / 28 | **0.857** |

- 常時噛み合い式（Constant meshed gears）
- アイドルギア（ドッグ結合していない空転ギア）はローラーケージ（needle bearing）で支持
- 変速機構：デスモドロミックドラム + シフトフォーク方式
- シフトペダル：左側蹴り込み方式

### ギア比ステップ（シフトアップ時の回転数変化率）

| シフト | 変化率 |
|--------|--------|
| 1→2 | 71.5%（回転数が約72%に落ちる） |
| 2→3 | 76.5% |
| 3→4 | 80.8% |
| 4→5 | 87.8% |
| 5→6 | 89.4% |

> ⚠️ **クイックシフター実装への注意**: 高速域のシフト（4→5、5→6）はギア比変化が小さいため、点火カット時間が短くても（50 ms程度）シフトが完了しやすい。低速域（1→2）は変化が大きく、確実な荷重抜きが必要。

## 3. 最終減速比（Final Drive Ratio）— 標準スプロケット

| 項目 | 仕様 |
|------|------|
| **フロントスプロケット（ドライブピニオン）** | **15T** |
| **リアスプロケット（クラウン）** | **37T** |
| **最終減速比** | **15/37 = 0.405（逆数2.467）** |

## 4. 総減速比（エンジン回転数→リアホイール）

各速でのエンジン回転数からリアホイール回転への総合減速比：

```
総減速比 = 一次減速比 × ギア内部比 × 最終減速比の逆数
         = 2.000 × ギア比 × (37/15)
```

| ギア | 総減速比 | 8,500 rpm時の車速参考（タイヤ周長約1.9m想定） |
|------|----------|----------------------------------------------|
| 1速 | 2.000 × 2.467 × 2.467 = **12.18** | 約 78 km/h |
| 2速 | 2.000 × 1.765 × 2.467 = **8.71** | 約 109 km/h |
| 3速 | 2.000 × 1.350 × 2.467 = **6.66** | 約 143 km/h |
| 4速 | 2.000 × 1.091 × 2.467 = **5.38** | 約 177 km/h |
| 5速 | 2.000 × 0.958 × 2.467 = **4.73** | 約 201 km/h |
| 6速 | 2.000 × 0.857 × 2.467 = **4.23** | 約 225 km/h |

> ※車速は概算。実際のタイヤ外径・スリップ・スプロケット変更で変化する。

## 5. ニュートラル検出スイッチ

- メインワイヤーハーネスに接続された**ニュートラルインジケータースイッチ**が装備されている
- ギアポジションセンサー（GPS）は非装備だが、ニュートラルスイッチの信号はArduinoでの状態管理に利用可能

> **クイックシフター実装への注意**: ニュートラルスイッチ（ONがニュートラル）をArduinoのデジタル入力に接続することで、ニュートラル時の誤作動を防止できる。1速→ニュートラルの誤シフト検出に有効。
