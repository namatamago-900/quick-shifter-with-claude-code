# その他・共通ドメイン知識 — Ducati 900SS (1995年式)

## 1. クラッチ仕様

### 形式

| 項目 | 900SS仕様 |
|------|-----------|
| **クラッチ形式** | **乾式クラッチ** |
| 操作方式 | **油圧式**（左ハンドルバーレバー → 油圧シリンダー） |
| 操作特性 | 軽め〜中程度の操作力。レバーフリーストローク 0.04〜0.06 in（1.0〜1.5 mm） |

> ⚠️ **乾式クラッチの特性（クイックシフター設計に重要）**:
> - 乾式クラッチはウェット式と比較して**ドラッグトルク（引きずりトルク）が非常に小さい**
> - シフトアップ時に駆動系の荷重抜きが行いやすい反面、急激な操作への応答が敏感
> - 点火カットによる荷重抜き時間は短くても有効
> - クラッチフルリリース不要のノークラッチシフトが実現しやすい構造
---

## 2. アイドル回転数（Idle RPM）

| 項目 | 値 |
|------|-----|
| **規定アイドル回転数** | **1,200〜1,300 rpm** |

> ⚠️ **注意**: アイドル回転数はクイックシフターのRPMスレッショルド設定の下限として使用する。1,200 rpm以下での点火カット作動は不要・有害。

## 3. 電装系（12V系統）

### 基本仕様

| 部品 | 仕様 |
|------|------|
| **バッテリー** | **12V - 16 Ah** (YUASA YB16AL-A2) |
| **オルタネーター（発電機）** | **12V - 300W** |
| **レギュレーター・レクチファイアー** | 30Aヒューズで保護 |
| スターターモーター | 12V - 0.7 kW |

### ヒューズ構成

| ヒューズ | 容量 | 回路 |
|---------|------|------|
| A1 | 30A | メイン（レギュレーター） |
| B2, C3 | 15A | 補助回路 |
| D4, E5 | 7.5A | 補助回路 |
| F6, G7, H8 | — | スペア（3本） |

### 充電システム動作特性

- バッテリー電圧が低い場合：充電電流 = **高（High）**
- バッテリーが充電済み（12〜14V）の場合：充電電流 = **4〜2A**
- 電気的安定性確認の目安：
  - ライトオフ（昼間）→ 約1,000 rpm でアンメーター指示ゼロ
  - スモールライトオン → 約1,200 rpm でアンメーター指示ゼロ
  - ヘッドライト（ロービーム）オン → 約2,200 rpm でアンメーター指示ゼロ

### クイックシフター電源設計への注意事項

1. **電圧変動**: アイドル時（1,200 rpm）付近では発電量が不安定になりやすい。Arduinoへの電源はバッテリーから直接取り、レギュレーターを別途使用することを推奨。
2. **ノイズ対策**: 点火コイルの一次側（0.34 Ω）は大電流スイッチングを行うため、電磁ノイズ源となる。Arduino入力ラインにはデカップリングコンデンサ（100 nF〜1 µF）の実装を推奨。
3. **エンジン停止時のバッテリー保護**: キーオフでバッテリーが急速に消耗する場合は接地不良を確認。

## 4. 1995年式900SS キャブレター車の固有特性

### インジェクション車との主な相違点

| 観点 | 1995年式 900SS（キャブレター） | 現代のインジェクション車 |
|------|-------------------------------|------------------------|
| 燃料供給 | MIKUNIキャブレター BDST 38-870/867 | 電子制御燃料噴射（EFI） |
| スロットル制御 | 機械式ワイヤー直結 | TPS（スロットル位置センサー）連動 |
| エンジンブレーキ制御 | なし（機械的のみ） | 燃料カット連動、スリッパークラッチ併用が多い |
| 点火制御 | KOKUSAN製固定進角カーブ | ECU制御マップ（多次元） |
| 始動補助 | チョーク（スターター弁）手動 | 電子制御コールドスタート |
| アフターファイア リスク | **高**（点火カット中も混合気供給継続） | 低（燃料カット連動） |
| RPMリミッター | 明記なし（KOKUSANモジュール内蔵の可能性） | ECU内蔵・可変設定可能 |

### キャブレター車でのクイックシフター実装時の固有リスク

1. **アフターファイア（バックファイア）**: 点火カット中もキャブレターから混合気が供給される。排気管内で未燃ガスが着火し、バックファイアが発生する可能性がある。**点火カット時間を必要最小限（50〜100ms）に抑えることが重要。**

2. **エンジンブレーキによる荷重変動**: TPSがないため、スロットル開度を電気的に検出できない。シフトセンサー（ストレインゲージ等）の信号のみでシフト操作を検出する設計が必要。

3. **低回転域での失火リスク**: キャブレターの負圧依存のため、低回転（1,500 rpm以下）での点火カットは失火→エンスト → **3,000 rpm以上を点火カット作動の下限として設定することを推奨**。

4. **エンジン回転数検出**: KOKUSANピックアップからの信号を直接利用する場合、信号電圧はバッテリー電圧に近い交流信号（AC）となる可能性がある。Arduinoの入力電圧（5V）に対するレベル変換回路が必要。
