# 点火系ドメイン知識 — Ducati 900SS (1995年式 キャブレター車)

## 1. 点火システム概要

| 項目 | 仕様 |
|------|------|
| 点火方式 | 電子式・誘導放電型（Inductive Discharge System） |
| 点火ユニットメーカー | **KOKUSAN**（国産電機） |
| ピックアップ方式 | パッシブ磁気ピックアップ（Reluctor / Variable Reluctance型） |
| ピックアップ数 | **2個**（水平シリンダー用・垂直シリンダー用、各独立） |
| ピックアップ抵抗 | 220 Ω ± 5% |
| コイル HV回路抵抗 | 3,700 Ω ± 10% |
| コイル LV回路抵抗 | 0.34 Ω ± 10% |
| プラグ（900SS） | CHAMPION RA6YC |
| プラグ抵抗キャップ | 10 kΩ（各シリンダー） |

---

## 2. フライホイール・ピックアップ構成（クイックシフター設計への重要注記）

本マニュアルには**フライホイールの歯数（パルス数/回転）は明記されていない**。

ただし以下が確認されている：

- ピックアップはフライホイール上の**参照マーク（刻印）に位置合わせ**する方式
- 組み付け時の手順：「水平シリンダーのピストンをTDC（上死点）に合わせ、ピックアップ参照マークをフライホイールの参照マークに一致させる」
- これはKOKUSAN製の典型的なシングル/デュアルパルス方式（クランク1回転あたり**各気筒1パルス**）と推定される
- 90° V型2気筒エンジンのため、クランク1回転で2パルス（各気筒TDC前の固定角度でトリガー）が出力されると考えられる

> ⚠️ **クイックシフター実装への注意**: パルス間隔はクランク1回転=2パルス（約180°間隔ではなく、Lツイン構造により不等間隔）。回転数の計算にはこの非対称性を考慮すること。実測による確認を推奨する。

## 3. 点火進角（Ignition Timing / Advance）

| 回転域 | 進角 |
|--------|------|
| アイドル〜約1,700 rpm ± 200 rpm | **6° BTDC**（固定） |
| 1,700〜2,600 rpm ± 300 rpm | 6° → 32° へプログレッシブ進角 |
| 2,600 rpm 以上 | **最大 32° BTDC**（固定） |

- 進角確認はストロボライトにて実施（左側カバーの透明点検窓より参照マークを目視）
- ユニットは全自動進角（機械式進角機構なし、電子制御）

---

## 4. 最大エンジン回転数（レブリミット）

| 項目 | 値 |
|------|-----|
| **絶対最高回転数（レブリミット）** | **9,000 rpm** |
| 最大出力発生回転数（900SS） | **8,500 rpm** |

> マニュアル明記：「**IMPORTANT - Under no circumstances the engine must go over 9000 rpm.**」

## 5. クイックシフター実装における点火カット時間の考察

### 点火カットの安全マージン

```
最大回転数: 9,000 rpm
クランク1回転の時間 = 60,000 ms / 9,000 = 6.67 ms/回転

進角が6°（アイドル）の場合：
  点火タイミング = TDCの6°前
  クランク6°の時間 = 6.67 ms × (6/360) = 0.11 ms

進角が32°（高回転）の場合：
  点火タイミング = TDCの32°前
  クランク32°の時間 = 6.67 ms × (32/360) ≈ 0.59 ms
```

### 推奨点火カット時間

- 点火カット時間（切断持続時間）は **50〜150 ms** 程度が一般的な目安
- 本車はキャブレター車のため、スロットルレスポンスがインジェクション車より緩やか
- 高回転域（8,000+ rpm）でのシフト時は短めのカット（50〜80 ms）が望ましい
- カット信号はコイルLV側（0.34 Ω、プライマリー）への電流遮断で実現可能

## 6. キャブレター車固有の電気的特性

| 観点 | 内容 |
|------|------|
| スロットル位置センサー（TPS） | **存在しない**（機械式スロットルバタフライのみ） |
| ECU / 燃料噴射制御 | **存在しない**（MIKUNIキャブレターBDST 38-870/867） |
| 点火マップ | 固定進角曲線（電子的リマッピング不可） |
| エンジンブレーキ特性 | インジェクション車より強め（燃料カットなし、負圧のみ） |
| 点火カット中の燃料供給 | キャブレターのベンチュリー効果により継続（インジェクション車のような燃料カット連動なし） |

> **重要**: キャブレター車では点火カット中もキャブレターから燃料は供給され続ける。エンジンブレーキ時の未燃混合気によるマフラー内爆発（アフターファイア）のリスクに注意。点火カット時間を必要最小限にすることが重要。
