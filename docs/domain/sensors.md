# センサー選定
クイックシフターを実現するために使用予定のセンサー類を記載する

## 1.シフトセンサー
ライダーのシフトアップを検知するために装着する
- ヤマハ YZF-R6 (2010年〜2016年) / YZF-R1 (2009年〜2014年)のシフトセンサー
    - 品番： 13S-82470-00（または 4C8-82470-00）
    - 単純なON/OFFを出力する仕様
    - 配線は2本出ている。それぞれ極性はなし。1つの配線をArduinoのデジタルピン([D5](../pin_assign.md))。もう一方をArduinoのGNDに接続予定
    - 純正シフトロッドを加工して装着 or 汎用のシフトロッドと組み合わせて装着予定
- ※上記以外のセンサー・スイッチの可能性もあり
    - 単純なON/OFFを出力するもの・他車種で使用されている純正品を使用予定

## 2.クラッチセンサー
ライダーのクラッチ操作を検知するために装着する
- DRC ブレーキプレッシャースイッチ バンジョーボルトタイプ (M10 P=1.0mm)
    - 品番：F5945
    - メーカー品番：D45-71-482
    - 出力は単純なON/OFF
    - 配線は2本出ている。それぞれ極性はなし。1つの配線をArduinoのデジタルピン([D6](../pin_assign.md))。もう一方をArduinoのGNDに接続予定
    - ducati 900ss 1995年式純正のクラッチマスターのバンジョーボルトを上記に交換して装着予定
    - ブレーキスイッチだが油圧を検知したいためクラッチに流用する
- ※上記以外のセンサー・スイッチの可能性もあり
    - 単純なON/OFFを出力するもの・他車種で使用されている純正品・その他汎用品を使用予定
