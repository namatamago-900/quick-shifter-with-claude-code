## ピンアサイン一覧

| ピン番号 (Nano) | 役割 | 入出力 | 接続先・加工箇所 | 備考 |
| :--- | :--- | :--- | :--- | :--- |
| **D3** | RPMパルス入力 | INPUT (Interrupt) | タコメータ | 外部割り込み(INT0)を使用 |
| **D5** | シフトロッドセンサー | INPUT | シフトロッドスイッチ（新規設置） | 押し/引き検知 |
| **D6** | クラッチスイッチ | INPUT | クラッチレバー配線（信号分岐） | 安全装置判定用 |
| **D8** | 点火カット指令(L) | OUTPUT | イグナイターL ～ コイルL（割り込み） | MOSFET経由で制御 |
| **D9** | 点火カット指令(R) | OUTPUT | イグナイターR ～ コイルR（割り込み） | MOSFET経由で制御 |
| **VIN** | 電源供給 | POWER | 車体12V | DC-DC等での安定化推奨 |
| **GND** | グランド | GND | 車体アース（フレーム/バッテリー） | 回路の基準電位 |
