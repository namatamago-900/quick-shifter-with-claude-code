## ピンアサイン一覧

| ピン番号 (Nano) | 役割 | 入出力 | 接続先・加工箇所 | 備考 |
| :--- | :--- | :--- | :--- | :--- |
| **D3** | RPMパルス入力 | INPUT (Interrupt) | ピックアップコイル(L or Rのどちらか) | 外部割り込み(INT0)を使用 |
| **D5** | シフトロッドセンサー | INPUT | シフトロッドスイッチ（新規設置） | シフトアップ検知 |
| **D6** | クラッチスイッチ | INPUT | クラッチレバー配線（信号分岐） | 安全装置判定用 |
| **D8** | 点火カット指令 | OUTPUT | イグニッションコイル一次側への12V電力供給ライン | 各コイルに分岐する前で加工する予定なので、D8ピンで足りるはず |
| **VIN** | 電源供給 | POWER | 車体12V | DC-DC等での安定化推奨 |
| **GND** | グランド | GND | 車体アース（フレーム/バッテリー） | 回路の基準電位 |
